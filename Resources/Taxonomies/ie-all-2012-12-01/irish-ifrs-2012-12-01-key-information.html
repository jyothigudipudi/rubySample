XBRL Ireland IFRS Taxonomy – 2012-12-01


Purpose and Scope

Represents reporting by ordinary commercial and industrial companies filing under IFRS

Owner

Business Reporting Ireland Limited (Trading as XBRL-Ireland)

Formal version date

1 December 2012

Version number

1.0

Schema location and reference

http://www.xbrl-ie.net/public/ci/2012-12-01/ifrs/core/2012-12-01/ie-ifrs-full-2012-12-01.xsd

Location of taxonomy zip file

http://www.xbrl-ie.net/public/ci/2012-12-01/ie-all-2012-12-01.zip


Viewing the taxonomy

This taxonomy is distributed by zip file which may be downloaded from the XBRL Ireland website at the address above.

The taxonomy should be viewed in appropriate XBRL software. To view or access the full contents, use the file
ie-ifrs-main-2012-12-01.xsd which is in the top-level folder ie-all-2012-12-01.
