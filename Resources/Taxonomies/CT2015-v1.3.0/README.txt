CT 2015 Computations Taxonomy version 1.3.0
Formal version date: 2015-02-01
Release date: 2015-10-16

Taxonomy package contents:
* CT2015-v1.3.0/
  * README.txt - this document;
  * .taxonomyPackage.xml - Taxonomy package file, including entry points and remappings;
  * remappings/remappingsCatalog.xml - OASIS remappings catalog;
  * www.hmrc.gov.uk/schemas/ct/ - CT taxonomy;
    * comp/ - CT computations taxonomy;
    * dpl/ - CT detailed profit and loss taxonomy;
    * combined/ - Combined entry points;
  * xbrl.iasb.org/ - Additional taxonomy included for convenience;
  * www.xbrl.org/ - Additional taxonomies included for convenience;
  * reports/
    * CT_2014_Schema_Locations_and_Normative_References.xls - Detailed entry-point information;
    * taxonomyStructure.csv - Table of all included taxonomy documents with normative locations and schema namespaces;
    * versionReport.xls - Version report (comparing to 1.2.0), human-readable;
    * versionReport.pdf - Version report (comparing to 1.2.0), human-readable;
    * TechnicalChanges-CT2015-v1.2.0-to-CT2015-v1.3.0.xls - Full technical difference report (comparing to v1.0);
    * ct-2015-02-01-pres-2015-09-09.xls - Rendering of the presentational structure of the taxonomy;
  * instances/
    * XBRL-sample-ct-comp-2015.xbrl - An XBRL instance for taxonomy validation only; includes at least one fact for each item changed within the taxonomy since previous version;
    * iXBRL-sample-ct-comp-2015.xhtml - An inline XBRL instance for taxonomy validation only; includes at least one fact for each item changed within the taxonomy since previous version.
